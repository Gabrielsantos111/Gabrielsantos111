# Olá, eu sou o Gabriel dos Santos Carvalho! (⌐■_■)

**Desenvolvedor de Software | Integração de Sistemas & Automação**

Sou um desenvolvedor recém-graduado apaixonado por transformar regras de negócio complexas em soluções de software eficientes e escaláveis. Meu foco principal é resolver problemas do mundo real através da automação de processos, construção de back-ends estruturados e integrações de APIs. 

Acredito que o melhor código é aquele que gera impacto direto e mensurável para os usuários e para o negócio.

 **O que eu faço:**
*   **Automação e Integração:** Construo APIs RESTful e Webhooks para conectar sistemas independentes (SaaS) e automatizar fluxos de trabalho.
*   **Back-End:** Desenvolvimento sólido utilizando Python, com foco em código limpo e arquitetura eficiente.
*   **Versatilidade Prática:** Experiência de ponta a ponta, desde a prototipagem de sistemas web de gestão até comunicação de redes distribuídas (P2P) e programação de baixo nível (IoT/Arduino).

 **Minha Caixa de Ferramentas:**
*   **Linguagens:** Python, C/C++
*   **Web & APIs:** Flask, RESTful APIs, Webhooks, manipulação avançada de JSON
*   **Infraestrutura & Ferramentas:** Git, GitHub, Render (Cloud Hosting), Google Sheets API
*   **Engenharia de Software:** Modelagem de requisitos (PM Canvas), documentação técnica e estruturação de banco de dados

 **Projetos em Destaque:** 

*    **API de Automação CRM para Delivery (SaaS):**
    Desenvolvimento de uma API em nuvem (Python/Flask) que intercepta webhooks de uma plataforma de vendas. O sistema calcula inteligência de estoque, gerencia cashback e atualiza painéis dinâmicos em tempo real para recuperação de clientes inativos e carrinhos abandonados.
*    **Sistema de Gestão Web para Barbearia (Full-Stack):**
    Planejamento ágil (PM Canvas) e desenvolvimento de ponta a ponta de uma aplicação web para o controle interno da operação. Back-end construído em Python (Flask) integrado a um banco de dados relacional MySQL, com a interface de usuário desenvolvida em HTML e CSS.
*    **Arquitetura de Rede Peer-to-Peer (P2P):**
    Desenvolvimento de um sistema de comunicação descentralizada entre nós utilizando Python 3.13.
*    **Automação em Hardware (IoT):**
    Projetos em C++ aplicados a microcontroladores, desenvolvendo lógicas de sinalização, reescrita de portas lógicas e controle de drivers de motor (L293D).
*    **CLINIX - Gestão Odontológica (Full-Stack):**
    Sistema voltado para a área da saúde e clínicas em geral. Interface construída com HTML/CSS e regras de negócio processadas no back-end com Python (Flask), integradas a um banco de dados MySQL para gestão segura de prontuários e agendamentos.
